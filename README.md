# Langpack Inspector

Inspect Ubuntu language packs and find missing translations.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install langpack-inspector
```
